## 📚 arXiv 논문


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.05400" target="_blank">OPUS: Towards Efficient and Principled Data Selection in Large Language Model Pre-training in Every Iteration</a></h3>



고품질 데이터가 고갈되는 현상에 대응하여, 학습 단계마다 최적화 도구(Optimizer)의 업데이트를 반영해 데이터를 선별하는 OPUS 프레임워크를 제안합니다. 기존의 정적인 필터링 방식과 달리 학습 역학을 실시간으로 고려하여 LLM 사전 학습의 효율성을 크게 높였습니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.09856" target="_blank">Code2World: A GUI World Model via Renderable Code Generation</a></h3>


<div class="categories">
<span class="category-tag">🧠 모델/아키텍처</span> <span class="category-tag">🤖 에이전트</span> 
</div>


GUI 에이전트가 미래를 예측하며 행동할 수 있도록, 렌더링 가능한 코드를 생성해 다음 화면을 시뮬레이션하는 Code2World 모델입니다. 기존 텍스트나 픽셀 기반 방식보다 훨씬 높은 시각적 정밀도와 구조적 제어 능력을 제공하여 사람과 유사한 상호작용을 가능하게 합니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.10063" target="_blank">Chain of Mindset: Reasoning with Adaptive Cognitive Modes</a></h3>


<div class="categories">
<span class="category-tag">🧠 모델/아키텍처</span> 
</div>


인간이 문제 해결 과정에서 여러 사고방식을 통합하듯, LLM도 단계별로 적절한 인지 모드(Mindset)를 전환하며 추론하는 새로운 방법을 소개합니다. 고정된 사고방식만 사용하는 기존 방식의 한계를 넘어, 문제 유형에 맞춰 유연하게 대처함으로써 해결 능력을 한층 끌어올렸습니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.09443" target="_blank">P1-VL: Bridging Visual Perception and Scientific Reasoning in Physics Olympiads</a></h3>


<div class="categories">
<span class="category-tag">🧠 모델/아키텍처</span> 
</div>


복잡한 다이어그램과 물리 법칙 이해가 필수적인 물리 올림피아드 수준의 문제를 해결하기 위해, 시각적 인식과 과학적 추론을 결합한 P1-VL 모델입니다. 추상적인 논리를 실제 물리적 현실과 연결하여 LLM이 과학적 일관성을 유지하며 추론하도록 돕습니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.10090" target="_blank">Agent World Model: Infinity Synthetic Environments for Agentic Reinforcement Learning</a></h3>


<div class="categories">
<span class="category-tag">🧠 모델/아키텍처</span> <span class="category-tag">🤖 에이전트</span> 
</div>


자율 에이전트 학습에 필요한 환경 부족 문제를 해결하기 위해, LLM을 활용해 다양한 합성 환경을 자동으로 생성하는 파이프라인(AWM)을 제안합니다. 이를 통해 일상적인 시나리오를 포함한 1,000개 이상의 환경을 구축하여 에이전트가 다양한 도구 사용법을 효과적으로 익히게 합니다.



</div>
