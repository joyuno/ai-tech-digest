</div>


실제 사용자 환경, 특히 차량 내 음성 비서와 같은 상황에서 LLM 에이전트의 일관성과 한계 인식 능력을 평가하는 CAR-bench를 소개합니다. 불완전하거나 모호한 명령이 주어지는 불확실한 상황에서 에이전트가 얼마나 안정적으로 대응하는지 측정합니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.05386" target="_blank">Spider-Sense: Intrinsic Risk Sensing for Efficient Agent Defense with Hierarchical Adaptive Screening</a></h3>


<div class="categories">
<span class="category-tag"><i data-lucide="brain"></i> 모델/아키텍처</span> <span class="category-tag"><i data-lucide="bot"></i> 에이전트</span> 
</div>


LLM 에이전트의 보안을 위해 모든 단계에서 강제적인 검사를 수행하는 대신, 내부적으로 위험을 감지하는 Spider-Sense 프레임워크를 제안합니다. 이를 통해 에이전트는 필요한 순간에만 선별적으로 보안 메커니즘을 작동시켜 효율성을 높일 수 있습니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.05261" target="_blank">Length-Unbiased Sequence Policy Optimization: Revealing and Controlling Response Length Variation in RLVR</a></h3>


<div class="categories">
<span class="category-tag"><i data-lucide="brain"></i> 모델/아키텍처</span> 
</div>


RLVR(검증 가능한 보상을 통한 강화학습) 과정에서 응답 길이가 길어지는 현상이 추론 능력 향상과 어떤 관계가 있는지 분석한 연구입니다. 응답 길이에 대한 편향 없이 추론 성능을 최적화하고 제어할 수 있는 새로운 정책 최적화 방법을 제시합니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.02474" target="_blank">MemSkill: Learning and Evolving Memory Skills for Self-Evolving Agents</a></h3>


<div class="categories">
<span class="category-tag"><i data-lucide="brain"></i> 모델/아키텍처</span> <span class="category-tag"><i data-lucide="bot"></i> 에이전트</span> 
</div>


에이전트의 메모리 관리를 고정된 규칙이 아닌 학습 가능한 '기술'로 정의한 MemSkill을 소개합니다. 긴 대화 기록 속에서도 에이전트가 스스로 정보를 추출, 정리, 삭제하는 법을 발전시켜 유연하게 기억을 관리하도록 돕습니다.



</div>


<div class="digest-item" markdown="1">

<div class="categories">
<span class="category-tag"><i data-lucide="brain"></i> 모델/아키텍처</span> <span class="category-tag"><i data-lucide="search"></i> 추론/검색</span> 
</div>
