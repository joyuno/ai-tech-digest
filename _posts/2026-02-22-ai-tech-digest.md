## <i data-lucide="book-open"></i> arXiv 논문


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.12675" target="_blank">SLA2: Sparse-Linear Attention with Learnable Routing and QAT</a></h3>



기존 SLA 모델의 수학적 불일치 문제를 해결하기 위해 학습 가능한 라우팅(Learnable Routing)을 도입한 SLA2를 제안한 논문이에요. 이를 통해 Diffusion 모델의 영상 생성 성능을 높이면서도 계산 효율성을 크게 개선했답니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.14296" target="_blank">AutoWebWorld: Synthesizing Infinite Verifiable Web Environments via Finite State Machines</a></h3>


<div class="categories">
<span class="category-tag"><i data-lucide="bot"></i> 에이전트</span> <span class="category-tag"><i data-lucide="code-2"></i> 개발 도구</span> 
</div>


실제 웹 데이터 수집의 어려움을 해결하기 위해, 유한 상태 기계(Finite State Machine)를 이용해 검증 가능한 웹 환경을 합성하는 AutoWebWorld 프레임워크입니다. 자율 에이전트가 더 저렴하고 정확하게 학습할 수 있도록 고품질의 데이터를 생성해 줘요.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.13515" target="_blank">SpargeAttention2: Trainable Sparse Attention via Hybrid Top-k+Top-p Masking and Distillation Fine-Tuning</a></h3>


<div class="categories">
<span class="category-tag"><i data-lucide="wrench"></i> 학습/최적화</span> 
</div>


Diffusion 모델 가속을 위해 Top-k와 Top-p 마스킹을 결합하고 Distillation 파인튜닝을 적용한 SpargeAttention2를 소개합니다. 훈련 가능한 Sparse Attention이 왜 훈련이 필요 없는 방식보다 더 높은 희소성(Sparsity)과 품질을 달성할 수 있는지 분석했어요.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.14979" target="_blank">RynnBrain: Open Embodied Foundation Models</a></h3>


<div class="categories">
<span class="category-tag"><i data-lucide="brain"></i> 모델/아키텍처</span> 
</div>


로봇과 같은 Embodied AI를 위해 인식, 추론, 계획 능력을 통합한 오픈소스 시공간 파운데이션 모델 RynnBrain입니다. 물리적 세계의 시공간 역학을 이해하고 다양한 실제 환경에서 자율적으로 작동할 수 있도록 설계되었어요.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.16855" target="_blank">Mobile-Agent-v3.5: Multi-platform Fundamental GUI Agents</a></h3>


<div class="categories">
<span class="category-tag"><i data-lucide="target"></i> 신뢰성/안전</span> <span class="category-tag"><i data-lucide="bot"></i> 에이전트</span> <span class="category-tag"><i data-lucide="code-2"></i> 개발 도구</span> 
</div>


데스크톱, 모바일, 웹 등 다양한 플랫폼을 지원하는 강력한 GUI 에이전트 모델인 GUI-Owl-1.5를 소개하는 논문이에요. 2B부터 235B까지 다양한 크기를 제공하며, OSWorld 같은 주요 벤치마크에서 최고 수준의 성능(SOTA)을 달성했답니다.



</div>
