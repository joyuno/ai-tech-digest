## <i data-lucide="book-open"></i> arXiv 논문


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.12675" target="_blank">SLA2: Sparse-Linear Attention with Learnable Routing and QAT</a></h3>



기존 SLA(Sparse-Linear Attention)의 한계인 휴리스틱 분할과 연산 오류를 해결하기 위해, 학습 가능한 라우팅을 도입한 SLA2를 제안합니다. 이를 통해 비디오 생성 모델의 속도와 성능을 더욱 효율적으로 개선했습니다.



</div>


<div class="digest-item" markdown="1">

<div class="categories">
<span class="category-tag"><i data-lucide="bot"></i> 에이전트</span> <span class="category-tag"><i data-lucide="code-2"></i> 개발 도구</span> 
</div>


웹 GUI 에이전트 학습을 위한 데이터 수집의 어려움을 해결하기 위해, 유한 상태 머신(Finite State Machine) 기반의 합성 프레임워크인 AutoWebWorld를 제안합니다. 검증 가능하고 제어 가능한 웹 환경을 무한대로 생성하여 에이전트의 신뢰성을 높여줍니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.14979" target="_blank">RynnBrain: Open Embodied Foundation Models</a></h3>


<div class="categories">
<span class="category-tag"><i data-lucide="brain"></i> 모델/아키텍처</span> 
</div>


인지, 추론, 계획 능력을 통합한 오픈소스 엠바디드(Embodied) 파운데이션 모델인 RynnBrain을 소개합니다. 실제 물리 환경의 시공간 역학을 반영하여 로봇이 환경을 이해하고 위치를 인식하는 능력을 강화했습니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.16705" target="_blank">Learning Humanoid End-Effector Control for Open-Vocabulary Visual Loco-Manipulation</a></h3>



휴머노이드 로봇이 다양한 물체를 조작할 수 있도록 돕는 새로운 시각 정보 기반 제어 패러다임인 HERO를 발표했습니다. 대규모 데이터 수집의 어려움을 극복하고, 로봇 팔(End-Effector) 제어의 일반화 성능을 크게 향상시켰습니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.13515" target="_blank">SpargeAttention2: Trainable Sparse Attention via Hybrid Top-k+Top-p Masking and Distillation Fine-Tuning</a></h3>


<div class="categories">
<span class="category-tag"><i data-lucide="wrench"></i> 학습/최적화</span> 
</div>


확산 모델 가속화를 위한 학습 가능한 Sparse Attention 기법을 연구하고, 기존 Top-k/Top-p 마스킹의 실패 원인을 분석했습니다. 이를 보완하는 하이브리드 마스킹과 Distillation Fine-Tuning을 통해 생성 품질을 유지하면서 희소성을 극대화하는 방법을 제시합니다.



</div>
