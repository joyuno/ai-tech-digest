## 📚 arXiv 논문


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2601.22027" target="_blank">CAR-bench: Evaluating the Consistency and Limit-Awareness of LLM Agents under Real-World Uncertainty</a></h3>


<div class="categories">
<span class="category-tag">🧠 모델/아키텍처</span> <span class="category-tag">🔍 추론/검색</span> <span class="category-tag">🤖 에이전트</span> <span class="category-tag">💻 개발 도구</span> 
</div>


이상적인 환경이 아닌, 차량 내 음성 비서처럼 불확실성이 높은 실제 환경에서 LLM 에이전트의 신뢰성을 평가하는 벤치마크인 CAR-bench를 소개합니다. 사용자의 모호한 요청에 대해 에이전트가 얼마나 일관성 있게 대처하고 스스로의 능력을 인지하는지 측정하는 데 초점을 맞췄습니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.05386" target="_blank">Spider-Sense: Intrinsic Risk Sensing for Efficient Agent Defense with Hierarchical Adaptive Screening</a></h3>


<div class="categories">
<span class="category-tag">🧠 모델/아키텍처</span> <span class="category-tag">🤖 에이전트</span> 
</div>


자율 에이전트로 진화하는 LLM의 보안 문제를 해결하기 위해, 외부에서 강제로 검사하는 방식 대신 내재적이고 선택적인 위험 감지 프레임워크인 Spider-Sense를 제안합니다. 계층적 적응형 스크리닝을 통해 에이전트가 스스로 위험을 효율적으로 감지하고 방어하도록 돕습니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.02474" target="_blank">MemSkill: Learning and Evolving Memory Skills for Self-Evolving Agents</a></h3>


<div class="categories">
<span class="category-tag">🧠 모델/아키텍처</span> <span class="category-tag">🤖 에이전트</span> 
</div>


기존의 고정된 메모리 관리 방식에서 벗어나, 정보를 추출하고 정리하는 과정을 학습 가능한 '스킬'로 정의한 MemSkill을 제안합니다. 이를 통해 LLM 에이전트가 다양한 상호작용 패턴에 맞춰 기억을 유동적으로 관리하고 긴 대화에서도 효율성을 높일 수 있게 합니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.05261" target="_blank">Length-Unbiased Sequence Policy Optimization: Revealing and Controlling Response Length Variation in RLVR</a></h3>


<div class="categories">
<span class="category-tag">🧠 모델/아키텍처</span> 
</div>


RLVR 학습 과정에서 모델의 응답 길이가 길어지는 현상이 추론 능력 향상과 관련이 있다는 점에 주목하여, 알고리즘별 길이 변화 패턴을 분석한 연구입니다. 응답 길이에 편향되지 않으면서도 추론 성능을 최적화하고 길이를 제어할 수 있는 새로운 방법론을 제시합니다.



</div>


<div class="digest-item" markdown="1">

<h3><a href="https://arxiv.org/abs/2602.06036" target="_blank">DFlash: Block Diffusion for Flash Speculative Decoding</a></h3>


<div class="categories">
<span class="category-tag">🧠 모델/아키텍처</span> <span class="category-tag">🔍 추론/검색</span> 
</div>


기존의 Speculative Decoding 방식이 여전히 순차적인 드래프팅에 의존한다는 한계를 극복하기 위해, 병렬 생성이 가능한 Diffusion 모델을 활용한 DFlash를 제안합니다. 이를 통해 초안 생성 속도를 획기적으로 높여 전체 LLM 추론 지연 시간을 줄이고 GPU 효율성을 개선했습니다.



</div>
